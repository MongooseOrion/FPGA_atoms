# 基于 FPGA 的以太网音频传输系统

项目使用的硬件：

  * es7243 ADC 音频数据采样模块
  * 紫光同创 PGL50H

## 输入输出参数

| 键 | 值 |
| :--- | :--- |
| 音频量化位深 | 16bit |
| 音频采样率 | 48kHz |
| 音频输入通道数 | 2 |
| 以太网音频数据传输格式 | `np.int16` |
| 以太网音频输出通道数 | 1 |

## 使用方法

将比特流烧录后，你需要将电脑的以太网适配器 IP 地址设置为 `192.168.0.3` ，然后运行 `./python/audio_play.py` 即可听到播放的音频。
