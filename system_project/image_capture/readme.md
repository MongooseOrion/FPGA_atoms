# 基于 FPGA 的图像采集系统

使用 FPGA 对单目摄像头数据进行传输，其中传输使用 UDP 千兆传输协议。

使用的平台：

  * 紫光同创 PGL50H-6FBG484
  * PDS 2022.2Lite
  * OV5640 双目摄像头

## 输入输出参数

| 键 | 值 |
| :--- | :--- |
| 摄像头输出分辨率 | $1280\times 720$ |
| 摄像头输出帧率 | 30FPS |
| 图像数据格式 | `np.uint8` |

## 使用方法

将比特流烧录后，你需要将电脑的以太网适配器 IP 地址设置为 `192.168.0.3` ，然后运行 `./software/udp_video.py` 即可显示画面。
